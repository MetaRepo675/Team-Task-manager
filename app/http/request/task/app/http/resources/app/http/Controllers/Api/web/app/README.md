# Team Task Manager

A professional team task management system with Laravel 11 and Tailwind CSS

## Features
- ✅ Full authentication (Laravel Breeze)
- ✅ Task management (CRUD) with policy-based access levels
- ✅ Repository + Service architecture for clean and testable coding
- ✅ Full API (RESTful) with versioning (V1)
- ✅ Form Request Validation with Persian messages
- ✅ Smart cache with Redis/Cache
- ✅ Feature and Unit tests with Pest/PHPUnit
- ✅ Responsive UI with Tailwind CSS
- ✅ Solar date support (Jalali)
